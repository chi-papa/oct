# Order Copy Bookmarklet

受注管理システムの詳細ページから、受注情報をワンクリックでクリップボードにコピーするブックマークレットです。

## コピーされる内容

| 項目 | 取得元クラス |
|------|------------|
| 受注番号 | `.pull-left li` |
| 注文者 | `.rms-content-order-details-contact-info-names`（1つ目） |
| 送付先 | `.rms-row-wrapper` 内の同クラス |
| 住所 | `.rms-row-wrapper` 内の `.address` |
| 電話番号 | `.rms-row-wrapper` 内の `.phone` |

取得できなかった項目は自動的にスキップされます。

## 使い方

### 1. ブックマークに登録する

1. `order-copy-bookmarklet.html` をブラウザで開く
2. 「コピー」ボタンでコードをコピー
3. ブックマークバーを右クリック →「ページを追加」
4. 名前に `受注コピー` などと入力し、URL欄にコードを貼り付けて保存

> ブックマークバーが非表示の場合は `Ctrl+Shift+B` で表示できます。

### 2. 実行する

1. 受注管理システムで対象の受注詳細ページを開く
2. ブックマークバーの「受注コピー」をクリック
3. 右上に「✓ コピーしました」と表示されたら完了
4. 貼り付けたい場所で `Ctrl+V`
